Este es la parte de desarrollo del proyecto de reconocimiento facial de Ingenieria de Software

1. Install Requests
    python -m pip show requests 
2. Install Os
    pip install os
3. Install CV2
    pip install opencv-python
4. Install Mediapipe
    pip install mediapipe
5. Install MySQLdb
    pip install MySQL
6. Install fpdf
    python -m pip install fpdf
